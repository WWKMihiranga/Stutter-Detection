# Stuttering Detection System

An interpretable, weakly-supervised system for detecting stuttering events in speech.

## Features
- Real-time stuttering detection
- Millisecond-level event localization
- Interpretable AI explanations
- Privacy-preserving (auto-deletes audio)

## Usage
1. Upload a 3-second audio clip
2. Click "Analyze Speech"
3. View results and visualizations

## Dataset

The dataset is not included in this repository due to size and privacy constraints.

Download instructions:
- https://github.com/apple/ml-stuttering-events-dataset

After download:
Place files inside `data/raw/`
